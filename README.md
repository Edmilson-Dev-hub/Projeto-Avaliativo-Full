# Galáticos Fitness

Aplicativo de gerenciamento de treinos desenvolvido como projeto acadêmico utilizando Flutter e Flask.

## Funcionalidades

* Cadastro de usuários
* Login de usuários
* Criação de treinos
* Cadastro de exercícios
* Exclusão de treinos e exercícios
* Histórico de treinos concluídos
* Persistência de dados local

## Deploy

A aplicação está disponível em:

https://galaticosfitness.vercel.app

## Observação

A versão disponibilizada na Vercel utiliza SharedPreferences para armazenamento local dos dados.
Essa abordagem foi adotada para permitir a demonstração completa do sistema em ambiente de hospedagem estática, sem necessidade de um backend ativo.
